# HitVote
HitVote is a WPF desktop application built with C# and .NET 8, designed as a song contest and survey platform. Users can browse a list of songs, vote for their favorite, and view a real-time ranking based on votes. The app features role-based authentication (Admin and User), secure password hashing with BCrypt, and data persistence using Entity 

Tech Stack: C# · .NET 8 · WPF · MVVM · Entity Framework Core · SQL Server · BCrypt · FluentValidation · Serilog · LiveCharts2
